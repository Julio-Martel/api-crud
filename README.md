# api-crud
Mi primera API CRUD creada con Node.js y Express.js, que permite crear, consultar, modificar y eliminar registros usando métodos HTTP y respuestas en JSON.
<br>
<h1>  Primera API CRUD desarrollada con Node.js y Express.js.</h1>
<p>
Este proyecto permite realizar operaciones básicas sobre datos (Crear, Leer, Actualizar y Eliminar) utilizando métodos HTTP como GET, POST, PUT y DELETE.
La API maneja datos en formato JSON y fue creada como práctica para comprender el funcionamiento de servidores, rutas, parámetros, middlewares y manejo de solicitudes en el backend.
</p>
